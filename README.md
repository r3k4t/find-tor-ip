# Find Tor IP

<h6>Author:RKT</h6>

Find your IP address from Tor Network.Its build on perl program.


### Tor Network(start) ###

<h6>Terminal Command :</h6>

+ sudo service tor start

### Terminal Command ###

+ git clone https://github.com/r3k4t/find-tor-ip.git
+ cd find-tor-ip
+ sudo perl find-tor-ip.pl
